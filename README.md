# Analisi-dei-Fattori-di-Rischio-del-Diabete
Questo progetto analizza i fattori associati alla presenza del diabete utilizzando tecniche di statistica descrittiva, inferenziale e predittiva. L'obiettivo è identificare le variabili correlate alla malattia e sviluppare un modello di regressione che stimi la probabilità che un individuo sia affetto da diabete. 
L'analisi è stata realizzata utilizzando Python per il preprocessing dei dati e R per le analisi statistiche e la modellazione.

Il dataset deriva dal Behavioral Risk Factor Surveillance System (https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset), un sistema di sorveglianza sanitaria sviluppato dai CDC (Centers for Disease Control and Prevention) degli Stati Uniti.
